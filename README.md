# jds-bansos

## At Glance
Implementasi challange/tugas aplikasi yang dapat digunakan untuk menginput data-data warga yang membutuhkan bantuan di era pandemi.

- Kelas Vue

## Features
- Vue 2

- Vue-Router, Vuex, Vue-Meta
- Tailwind
- Styling berdasarkan Jabar Design System
